Modificación de árbol.

Modifica el árbol añadiendo un nuevo nodo en el sitio de un nodo determinado.

Precondición
Entran como parámetros un id de prioridad y un entero vacío auxiliar. Y como parámetros implícitos un árbol(principal) a modificar y un árbol auxiliar.
Postcondición
El árbol auxiliar es añadido como nodo del árbol principal si se cumple que del nodo modificado no salen otros nodos. En caso de que el procesador contenga procesadores auxiliares, no se modifica el árbol principal.

Alta de un proceso en un procesador.

Da de alta un proceso en un procesador específico.

Precondición
Lectura de un id de procesador y un proceso.
Postcondición
Hace un recorrido en el cluster en busca de un procesador con el mismo identificador. Si lo encuentra y existe algún hueco con un espacio suficiente o no existe un proceso con el mismo id, se añade el proceso y devuelve 0. En el caso que no exista un procesador con ese id devuelve 1. Si hay un proceso con el mismo id en el procesador devuelve 2. Si no hay espacio suficiente en la memoria devuelve 3.
